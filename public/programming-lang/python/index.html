Welcome to Python tutorial

Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.

Python was created way back in 1991, but that doesn't make it any obsolete! It is now one of the most popular programing language in the world.

Python is often described as a "batteries included" language due to its comprehensive standard library.

In this journey we will be helping you to learn the fundamentals and the basics of Python along the way!

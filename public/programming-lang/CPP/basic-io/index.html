C++ I/O Basic

This tutorial will guide you everything about C++ I/O Basic

In computer programming, Input/Output is the communication interface between information processing system and an information source, which could be a human, or another information processing system



C++ Output

  • In C++, we can use the object called std::cout to print(output) text, values,... etc etc
  • For example
    • #include <iostream>
    int main() {
    int points = 78;
    std::cout << "You have " << points << "points";
    return 0;
    }
  • Output:
    • You have 78 points

Explaining

  1. The #include <iostream> is to include the iostream header file so we can use the input/output feature.
  2. In main function, we have declared a variable called point and then assign it to 78 as a value with an integer type.
  3. Then we use std::cout (the cout object which is in the namespace called std) to print the string inside the two quotation marks, as well as the value of the variable. Each value is followed together with the << operator
  4. return 0; is used to end the main() function, this can be also called as the exit status

    5. Note that the variable does not need to use quotation marks

C++ Input

  • We can send (input)information to our program by using the function called std::cin. For short, we can use it to read user input.
  • For example:
    • #include <iostream>
    int main() {
    int salary;
    std::cin >> salary;
    std::cout << "Your salary is: " << salary;
    return 0;
    }
  • Output:
    • 15000
    Your salary is: 15000

Explaining

  1. In the above program, we have created a variable called salary without declaring a value. Then we used the input statement called std::cin to read user input
  2. std::cout to output the string in the quotation marks as well as the value followed by the << operator

Pro tip: You can try to add this before std::cin statement for better reading experience!

std::cout << "Please enter your salary";

Output:

Please enter your salary: 15000
Your salary is: 15000

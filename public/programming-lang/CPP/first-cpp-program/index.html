Your first C++ program and syntax

In this tutorial, we'll be helping you understand your first C++ program and syntax. However, before getting started. You need to do 3 things below:

  1. You need to install a text-editor. There are many of them, but most popular is Visual Studio Code. If you don't like it, you can try with Notepad, Sublime Text, etc etc.
  2. You will need to install a GCC compiler. to compile the C++ code to machine language.You can try with LLVM, GCC, MSVC, Clang,....

Your first C++ program

Create a cpp file with ".cpp" at the end of the name. You can name it with every name you want.

Now write the following C++ code and save it:

#include <iostream>
int main() {
std::cout << "Hello World!";
return 0;
}

Explaining

  1. the #include <iostream> is to include the header file library. In this case, iostream is the library that let us work with input and output objects. Such as std::cout. Header files provides a lot of functionality for a C++ program.
  2. this is just a line space and optional. It makes the code more readable.
  3. A C++ program must have a main function. The curly brackets indicate the start and the end of the main function.
  4. std::cout prints the string inside the quotaion marks. std::cout must be followed by a << and after that is a format string. In this case, "Hello World" is a format string
  5. The return 0; statement is the exit status of the program. Which means that the program ends with this statement. return 0; means success. and 1 means failure.

Note:

  • Every C++ statements ends with a ";" semicolon.
  • The compiler ignores the white space in the code.
  • You can actually write your main() code in just 1 line. Just remember to put a ";" in each statement!

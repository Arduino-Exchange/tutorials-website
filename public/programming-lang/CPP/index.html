Welcome to C++ tutorial

C++ is a superset programming language of C created by Bjarne Stroustrup in 1985.

C++ is one of the most popular language in the world. It is a statically compiled programming language and was used for many applications like software dvelopment, game development, operating systems and much more.

C++ is like C, but supports a lot of features. Like OOP(Object Oriented Programming), procedural and so on. It is also really powerful.

In this journey we will be helping you to learn the fundamentals of C++ along the way!

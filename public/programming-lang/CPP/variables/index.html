C++ Variables and Data Types

This tutorial will assist you in understanding C++ variables, data types.

In computer science (programming), a variable is a container that can hold, store and change data.

In C++, there are numerous amount of data types. And they are defined with different keywords below:

  • int - which stores integers(without decimals) from -2147483648 to 2147483647 in values
  • float - which stores floating-point values from 3.4E38 to 3.4E-38. It is used for decimal and exponential values.
  • double - which stores floating-point values from 1.7E308 to 1.7E-308. Which also used for decimal and exponential values
  • char - which stores only single characters(letters), such as 'e', 'D', '7', 'y' and so on.
  • bool - which stores values with only two states: "true" and "false".
  • string - stores text, such as "hello there!", "wassup",...

These are some basic variables in C++ programming. Now let's learn how to use them!

To initialize a variable, specify the type and assign it with a valid value. Below is the syntax:

type variable_name = value;

Here, type stands for any specific variables. variable_name stands for any name, such as "myName", "age", "mark",... And value must be a valid value corresponding to data types

For example, to create an integer name's "myVar" that stores a value of 100, we can do:

int myVar = 100;

You can create a variable without declaring a value, then create it later like below:

int myVar;
myVar = 100

You can change the data(value) of the variable. However, if you assign a new value to an existing value of the variable. The previous variable would be lost

int myVar;
myVar = 100;
std::cout << myVar; //Outputs 100
myVar = 69420;
std::cout << myVar; //Outputs 69420

Assigning variables

You can assign variables together to form another value that might be useful in some programs.

int x = 69;
int y = 420;
int sum = x + y;

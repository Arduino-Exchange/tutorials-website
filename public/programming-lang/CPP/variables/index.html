C++ Variables and Data Types

This tutorial will assist you in understanding C++ variables, data types.

In computer science (programming), a variable is a container that can hold, store and change data.

In C++, there are numerous amount of data types. And they are defined with different keywords below:

  • int - which stores integers(without decimals) from -2147483648 to 2147483647 in values, which takes up about 4 bytes in size
  • float - which stores floating-point values from 3.4E38 to 3.4E-38. It is used for decimal and exponential values, which takes up about 4 bytes in size
  • double - which stores floating-point values from 1.7E308 to 1.7E-308. Which also used for decimal and exponential values but has a bigger size than float, takes up about 8 bytes
  • bool - which stores values with only two states: "true" and "false". It has 1 byte of size
  • char - which stores only single characters(letters), such as 'e', 'D', '7', 'y' and so on. It also has 1 byte of size
  • string - stores text, such as "hello there!", "wassup",... This data type has 24 bytes in size.
  • void - this is considered a data type. However, it is basically a keyword to use as a placeholder... For short, this represents as "no data"

These are some basic variables in C++ programming. Now let's learn how to use them!

To initialize a variable, specify the type and assign it with a valid value. Below is the syntax:

type variable_name = value;

Here, type stands for any specific variables. variable_name stands for any name, such as "myName", "age", "mark",... And value must be a valid value corresponding to data types

For example, to create an integer name's "myVar" that stores a value of 100, we can do:

int myVar = 100;

You can create a variable without declaring a value, then create it later like below:

int myVar;
myVar = 100 // Assigning myVar to 100

You can change the data(value) of the variable. However, if you assign a new value to an existing value of the variable. The previous variable would be lost

int myVar;
myVar = 250; // Assigning myVar to 100
std::cout << myVar; // outputs 250
myVar = 69420; // Assigning myVar to 69420
std::cout << myVar; // outputs 69420

Assigning variables

You can assign variables together to form another value that might be useful in some programs.

int x = 69; // Assigning x to 69
int y = 420; // Assigning y to 420
int sum = x + y; // sum = 489



Variables Reference

1. Integer(int)

  • int keyword is used to represent integer numbers
  • Declaring example:
    • int income = 783000;

2. float and double

  • float and double are used to represent floating-point numbers, such as: 13.56, 96.69,... etc etc.
  • float has 4 bytes and double, as it name implies, it has 8 bytes of size
  • float has 7 decimal digits of precision and occupies 32 bits. doubleis a 64-bit IEEE 754 double-precision floating-point number. which means, double is more precise and also for storing larger value.
  • Declaring example:
    • double points = 65.852;
    float kilograms_of_sugar = 152.7;

3. Boolean(bool)

  • bool is used for conditional statements in loops, expressions, algorithms,... etc etc.
  • bool has only 1 byte of size. It only stores two values: true and false.
  • Declaring example:
    • bool earthIsFlat = false;
    bool condition = true;

4. Character(char)

  • char is used to store characters, but only 1 character at once.
  • It has 1 byte of size.
  • char in C++ is enclosed by using 2 quotation mark outside the character value.
  • Declaring example:
    • char character = 'e';

5. string

  • string or to be more exact, std::string, is used to store an array of characters. Like: "Hello World", "password",... etc etc
  • It has 24 bytes in size.
  • Declaring example:
    • std::string = "Programming Is Fun!";
    std::string = "C++";
    std::string = "Computer Science";

6. void

  • void keyword is used to represent as "no data" or "nothing"
  • void keywords are widely used for functions, procedures and pointers
  • Declaring example:
    • void main();
    void calculate();
